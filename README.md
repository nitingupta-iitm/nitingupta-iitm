## 🧑‍💻 About Me

I'm a data science student at **IIT Madras (BS Programme)** with a strong focus on **Generative AI**, **Machine Learning**, and **data-driven decision-making**. I enjoy building end-to-end solutions — from raw data pipelines to production-ready AI applications — and I'm especially passionate about leveraging LLMs and the Claude API to create powerful, practical tools.

- 🎓 Currently pursuing **BS in Data Science & Applications** at IIT Madras
- 🤖 Actively building with the **Anthropic Claude API** — prompt engineering, RAG, tool use, agents & MCP
- 📊 Experienced in turning messy business data into actionable insights (30–40% sales uplift for a real business!)
- 🔭 Exploring **agentic workflows**, **GenAI-powered analytics**, and **full-stack AI applications**
- 📍 Based in India
